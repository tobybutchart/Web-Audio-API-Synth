## Description
Synth made from Web Audio API. Using [Tuna.js](https://github.com/Theodeus/tuna), [Kontrol.js](http://anthonyterrien.com/kontrol/) and [Qwerty.js](https://stuartmemo.com/qwerty-hancock/)
